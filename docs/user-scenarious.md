# Сценарій користувача (MVP)

**Умова:** користувач відкрив додаток "KAVINKA"

**Кроки:**
1. Відкрити додаток.  
2. Переглянути каталог продукції.  
3. Додати каву та десерт до кошика.  
4. Перейти до оплати та сплатити онлайн.  
5. Отримати сповіщення про статус замовлення: "Готується" → "Готово до видачі".  

**Очікуваний результат:** замовлення підтверджене, користувач бачить актуальний статус і може забрати продукцію без проблем.

## Додаткові сценарії (опційно для наступних ітерацій)
- Повторне замовлення улюблених продуктів через історію замовлень.  
- Використання бонусів та акцій у програмі лояльності.  
- Відстеження найближчих кафе KAVINKA на карті та очікуваного часу приготування.
