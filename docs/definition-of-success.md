# Критерії успіху для MVP

1. **Онлайн-замовлення:** користувач може зробити замовлення через додаток без помилок.  
2. **Оплата:** замовлення успішно оплачено онлайн.  
3. **Статус замовлення:** користувач бачить актуальний статус ("Готується", "Готово до видачі") в реальному часі.  
4. **Стабільність:** додаток працює без критичних збоїв протягом щонайменше 7 днів використання.  
5. **Простота використання:** новий користувач може пройти головний сценарій без допомоги або інструкцій.

## План перевірки

- Провести тестування головного сценарію із 5-10 новими користувачами.  
- Перевірити оновлення статусу замовлення у реальному часі.  
- Переконатися, що всі функції MVP працюють без багів на різних пристроях.  
- Зібрати зворотний зв'язок щодо UX/UI та часу виконання замовлення.
